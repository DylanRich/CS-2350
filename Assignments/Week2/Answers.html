What is the difference between TCP/IP and HTTP?

TCP/IP refers to the transfer and addressing infrastructure that makes up the internet. It is the manner in which devices and networks connect to the internet, identify websites, and access them. HTTP refers to the markup language that web browsers interpret in order to display webpages

Define the term: URL:

A URL, or a Uniform Resource Locator, also called a web address, refers to the unique sequence of characters which identify a website on the internet and can be used to access it.

What is the Client/Server Model?

The Client/Server Model is the basic model on which most web services and websites are basesd. It consists of a central server which contains all of the data for the service or website, and client systems or networks which request that data from the server.

What two things does the broser do?

The web browser requests the data of a website from the server, and then compiles it together into the webpage you actually see.
